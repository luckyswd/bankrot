# Legal Docs

## Структура проекта

```
test_3/
└── frontend/          # React приложение
    ├── src/
    │   ├── components/
    │   │   ├── ui/           # UI компоненты (Shadcn)
    │   │   ├── databases/    # Справочники
    │   │   ├── Login.jsx
    │   │   ├── Dashboard.jsx
    │   │   ├── ClientCard.jsx
    │   │   ├── DocumentsPage.jsx
    │   │   └── Layout.jsx
    │   ├── context/
    │   │   ├── AppContext.jsx
    │   │   └── ThemeContext.jsx
    │   ├── data/
    │   │   └── mockData.js
    │   └── lib/
    │       └── utils.js
    ├── package.json
    ├── vite.config.js
    ├── tailwind.config.js
    └── README.md
```

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Откроется на http://localhost:3000
