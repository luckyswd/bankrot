# Legal Docs - Система управления юридическими документами

Приложение для автоматизации работы с юридическими документами.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откроется на http://localhost:3000

## Сборка

```bash
npm run build
```
